# PlayBox - Next.js + Tailwind demo (Roblox-inspired)

Aceasta este o versiune completă a demo-ului, pregătită pentru push pe GitHub și deploy pe Vercel.

**Conținutul pachetului**
- Next.js app (pages/)
- Tailwind configurat (tailwind.config.js, postcss.config.js)
- Scripturi în package.json

**Cum folosești local**
1. `npm install`
2. `npm run dev`
3. Deschide http://localhost:3000

**Deploy pe Vercel**
1. Pushează acest repo pe GitHub.
2. În Vercel, alege "Import Project" și conectează repo-ul GitHub.
3. Vercel detectează Next.js și face deploy automat.
